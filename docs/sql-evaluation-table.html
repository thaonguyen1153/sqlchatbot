How to use

  1. Run each question through the application.
  2. Paste the SQL, result notes into the row.
  3. Pick values for SQL validity, result correctness, answer correctness, and pass/fail.
  4. Enter response time in seconds.
  5. Read the live metrics at the top and bottom of the page.

Metric formulas

  1. Pass rate = Passed tests / Total tests × 100
  2. SQL validity = Valid SQL queries / Total tests × 100
  3. Result accuracy = Correct result sets / Total tests × 100
  4. Answer accuracy = Correct answers / Total tests × 100
  5. Edge-case success = Passed edge cases / Total edge cases × 100
  6. Average response time = Total response seconds / Timed tests
